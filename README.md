# CRESt
Copilot for Real-world Experimental Scientist (CRESt), an automated robotic platform for conducting electrochemical reactions at MIT
The active learning algorithm files could be located in active_learning.
The hardware control includes the control of the Opentrons liquid handling robot in liquid_handling_robot, Xarm for automated electrochemical testing in robotic_testing, automated SEM control in crest, and database recording in db_control.
